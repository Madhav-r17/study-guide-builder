# AI-Powered Interactive Study Guide Generator

## Overview
A full-stack web application that converts study material into AI-generated notes and study guides.

## Tech Stack
- React + Vite
- Node.js + Express
- MySQL
- Tailwind CSS
- AI API

## Features
- Upload study material
- Generate AI study guides
- Save and organize notes
- Responsive UI
